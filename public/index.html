Transform : Translate

En este ejemplo tienes una animación basada en transformaciones y traslaciones.

La transformación indica el efecto que se aplicará

Y traslete es el valor que en este caso indica cuantas posiciones debemos movernos en los eje X e Y.

transform: translate(20px, 20px);

Transform rotate, scale, skew y matrix

Los siguientes efectos pueden recibir varias propiedades en paralelo para lograr efectos mas alla del movimiento recto.

scale: Cambia el tamaño del objeto => transform: scale(150%);

skew: Desplazamientos en una dirección mas que en otra, muy util para los efectos de puerta o una imagen en diagonal.

Por defecto el efecto se aplica en el eje X pero podemos agregar el eje X o Y si deseamos indicar la rotación. transform: skewY(20deg);

rotate: Rotar un objeto ciertos grados

matrix: Este efecto es una combinación de scale y sckew Los objetos se mueven similar a una puerta y cambian su tamaño.

Transform Origin

Ayuda a redefinir el punto de rotación del objeto. Por defecto se define su centro como eje de rotación.

transform-origin: top right;

El origen debe declararse antes del disparador del evento.

Transform style y perspective

Es el primer acercamiento a los efectos en 3D.

Esta propiedad es similar a ver una imagen desde otra posición. El efecto de perspectiva inicia dandole una profundidad al elemento contenedor. Podemos decir que tendra profundidad en el momento de aplicar el efecto.

Ejemplo

Es similar a ver una ventana que se abre desde uno de sus bordes

El origen debe declararse antes del disparador del evento.

Pasos:

  • Crear un contenedor y un hijo
  • El contenedor debe tener la propiedad position relative
  • El contenedor define cual es su profundidad con perspective
  • El hijo debe definirse como position absolute
  • En el hijo en el disparador definimos transform-style: preserve-3d; para indicar que se movera en un movimiento 3D.
  • Despues debemos definir cual es el movimiento que ejecutara
  • En este ejemplo: transform: rotateX(50deg);

Backface visibility

Este efecto lo usamos para mostrar una imagen o una card que al disparar un efecto gire y veamos la información que esta en la cara posterior.

Requiere minimo de un contendor y dos hijos que seran las caras de nuestra card.

  • La card o contenedor debe tener position: relative; y transform-style: preserve-3d;
  • Los dos hijos deben tener la propiedad backface-visibility: hidden; para ocultar la cara al rotar
  • El efecto de rotar se aplica sobre el contenedor
  • .seccion9-card:hover{ transform: rotateX(180deg); transition-duration: 0.5s; transition-timing-function: ease-in; }
  • Y luego se define sobre el elemento hijo de la cara oculta
  • .face.back{ background-color: rgb(205, 255, 219); transform: rotateX(180deg); }
  • De esta manera rotan en direcciones opuestas para crear ele efecto
